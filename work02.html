The description of this work is found on the following link. In this page, I will show you two charts initially made in the Tableau. The data were exported with the own software (Tableau), saved in an excel file and after that manually inserted at the code javascript with highcharts.


First chart, in this chart I tried answer the follow question: What are the top ten films that had higher worldwide gross. At this chart, I used a few configurations in this tutorial.

Second chart, in this chart I tried answer the follow question: What the relationship between the films that won oscar awards and bafta awards. At this chart, I used line charts to compare the films that received any of these awards.

Conclusion, I had to search a few highcharts settings, but I could reproduce almost a hundred percent the same charts produced by Tableau without difficulties.
